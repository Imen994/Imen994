##  Featured Projects

### 📦 YUCCA ERP Mobile *(Final Year Project — PFE)*
> Flutter · FastAPI · Django · LightGBM · Docker · Supabase · Firebase FCM

Full ERP mobile application for managing quotes, orders, and products, with an embedded AI module for stock management.
- Designed MVVM architecture across the Flutter app
- Built an AI pipeline using **LightGBM** to classify stock alerts (critical / low) based on historical data
- Deployed the AI service via **FastAPI** in a **Docker** container
- Integrated **dual-channel FCM push notifications** (ERP alerts + AI alerts)
- Connected to a **Django REST Framework** backend with PostgreSQL/Supabase
